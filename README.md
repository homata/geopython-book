Pythonではじめる地理空間情報
=====
この資料はPythonを使って地理空間情報について記述をしています。

# 対象読者

このドキュメントは、次の方にお読みいただくことを前提に説明しています。<br>

* Pythonでプログラミング言語を学びたい方
* HTML, CSS, JacaScriptはある程度理解している方
* 地理空間情報のプログラミングは初めてという方
